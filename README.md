# Anpharos
The repo for the final project of "Estructuras".


## Building
For compiling, run the following command from the project root:

### UNIX (Linux & MACOS)
```
javac -cp . anpharos/Main.java
```

### Windows
```
javac -classpath . anpharos\Main.java
```

## Usage
For running, use:

### UNIX
```
java anpharos.Main
```

#### Windows
```
java -classpath . anpharos.Main
```
